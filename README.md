## 錦糸町プラン検索アプリ
全ユーザー  
- 錦糸町の施設を予算を指定して調べることができる。
- また、予算を指定して予算以下に収まるようにランダムで3店舗紹介する。

ログインユーザー  
- 錦糸町の店舗登録・削除・編集をできるようにする。
- 気に入った店舗をお気に入り登録をして一覧表示できるようにする。
- お気に入り登録した店舗から複数店舗を選びおすすめのプランとして登録できるようにする。
## 開発環境
- PHP 8.0.23
- Laravel 6.x
- HTML
- CSS
- MariaDB
- Cloud9
